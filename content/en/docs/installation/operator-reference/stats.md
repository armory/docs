**spec.spinnakerConfig.config.stats**

```yaml
stats:
  enabled: true
  endpoint: # Set the endpoint for stats metrics.
```

- `enabled`: true or false.
- `endpoint`: Set the endpoint for stats metrics, such as `https://stats.spinnaker.io`
